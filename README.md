# Gym_Website

## Overview

**Gym_Website** is a static website project created to represent a gym or fitness center.  
The project demonstrates fundamental front-end web development skills using HTML and CSS.

It can be used as a portfolio project or as a base for a real gym promotional website.

---

## Tech Stack

- HTML5
- CSS3

---

## Features

- Gym landing page design
- Sections such as About, Services, Programs, and Contact
- Media content (images and/or video)
- Clean and structured layout
